# DC-DC-Buck-Converter-based-on-the-Texas-Instruments-LMR51450
This project converts a 10–35 V DC input into a regulated 7.6 V / 4 A output. The design was developed by following the LMR51450 datasheet, selecting external components based on design calculations and TI's recommended application circuit, while applying standard PCB layout practices for switching power supplies.
